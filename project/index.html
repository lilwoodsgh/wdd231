Site Name

Site Name: EduConnect

This name was chosen to represent a centralized system for managing school-related information, including student data, staff communication, and reports.

Site Purpose

The purpose of the site is to provide a hub for school administrators and teachers to manage student profiles, communicate effectively, and generate reports related to school performance.

Scenarios

Scenario 1: A teacher wants to quickly look up a student's profile to check their grades and attendance history.

Scenario 2: An administrator needs to send out a message to all teachers about a school-wide event.

Color Schema

Primary Color: #1A237E (Dark Blue) – Used for headers, navigation bar, and call-to-action buttons.

Secondary Color: #F5F5F5 (Light Gray) – Used as the background color for the main sections of the page.

Accent Color: #4CAF50 (Green) – Used for buttons or highlighted actions like "Send Message" or "Generate Report".

Typography

Font 1: 'Roboto' – Used for general body text, menus, and navigation.

Font 2: 'Lora' – Used for headings and titles to create a clear contrast with the body text.

Wireframe

Desktop View

The desktop view will have the following structure:

  • Header: Logo on the left, navigation bar in the middle, and user profile on the right.
  • Main Content: Dashboard overview on the left side with stats and recent updates; Sidebar with quick links on the right.
  • Footer: Contact information and quick links to terms of service and privacy policy.

Mobile View

The mobile view will have a simplified layout with:

  • Header: Logo centered, hamburger menu on the left, user profile on the right.
  • Main Content: Dashboard overview stacked vertically with key stats and actions. Sidebar menu collapsed.
  • Footer: Simplified footer with links to contact info and policies.
